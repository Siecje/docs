Node.js integration
====================

Check the following articles to find the necessary information on integration with Node.js.


###Getting Started

Node.js integration is possible with Webix [data components](desktop/components.md), client-side datastores (called [Data Collections](desktop/nonui_objects.md)) and [forms](desktop/form.md).
It allows managing the create/save/update/delete operations. Besides, it provides dynamic loading for Tree, as well as sorting and filtering for all components.

- Supported databases: MongoDB, Mysql, Microsoft SQL Server, PostgreSQL.

List of data types
==================

Next data types are used through API reference

### Base types

- any			-	any kind of value can be used
- array
- bool
- date		-	js Date object instance
- element		-	DOM element
- hash		-	js object, used as hash
- id			-	string or number, which is used as ID
- object		-	js object
- view		-	view configuration
- function	-	JavaScript function


"object" is used for data items, different components or helpers objects

### Common mixed types

- bool,string		-	option can use true|false, or separate string defined modes
- bool,string,function	-	as above one, but for more flexibility custom method can be defined 
- id,element		-	id of html element, or element itself
- string,function	- 	string defined mode, or custom function
- string,view		-	text content or sub-view configuration

Специфика API
-------------

#### Страницы компонент и модулей

- автогенерируются фрейдом
- описания методов - из @short секции на страницах методов
- заголовок берется из api/obj/{name}.md

например - [datatable](api/obj/ui.datatable.md), код из memo идет в краткое описание , все остальное на страницу компоненты




### Страницы методов

в качестве примера можно глянуть 

[attachEvent](api/link/ui.datatable_attachevent.md)

код страницы разбит на секции, наиболее критичные из которых short, params - все остальные опциональные

иногда страница параметров при открытии будет показывать два редактора - один для базовой имплементации метода, второй конкретно для текущей компоненты.


Прочие операции
---------------

normalize.php - пересоздание оглавлений
validate.php - проверка структуры
export.php - генерирует статическую версию 
chm.php - генерирует chm файл

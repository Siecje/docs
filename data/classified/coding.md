Общие требования к коду
=======================

Код хранится в git - *git@github.com:Stanislav-Wolski/Webix4.git*, также есть [веб-интерфейс][gitrepo]

Коммиты в мастер *запрещены*, но если изменения не затрагивают код (поправленный семпл, орфография в документации и т.п.) - можно коммитить прямо в мастер, чтобы не было лишней бюрократии. 


JS код
------

- никаких глобальных переменных за пределами неймспейса webix
- комментарии в сложных местах приветствуются 
- использование classified/jslint.md - настоятельно рекомендуется
- использование [ассертов](classified/assert.md) приветствуется
- приватные методы начинаются с "_"
- паблик методы - пишутся  в camelCase


PHP код
-------

Поддерживаемая версия пхп - 5.2  

 - для операций с данными используются коннекторы
 - где возможно используем sqllite и включаем ее в гит
 - где надо дин. загрузка ( и sqllite не подходит ) - используем mysql на 192.168.3.251 ( user: datatable, pass:datatable )
 - общую конфигурацию ( коннект к базе, подключение библиотек ) - имеет смысл выносить в samples/common/

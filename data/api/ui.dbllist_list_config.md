list
=============

@short:
	sets the configuration of lists in the Double List widget

@type: object
@example:
webix.ui({
  view:"dbllist",
  list:{ autoheight: true},
  data:[
     {id:"1", value:"Contacts"},
     {id:"2", value:"Products"},
     {id:"3", value:"Reports"}
  ]
});



@template:	api_config

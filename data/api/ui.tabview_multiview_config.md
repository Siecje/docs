multiview
=============

@short: allows to configure multiview part of tabview
	

@type: object
@example:

~~~
webix.ui({
	view:"tabview",
    multiview:{
      animate:{ type:"slide" }
    },
    cells:[
      { header:"A", template:"My Tab 1 "},
      { header:"B", template:"My Tab 2 "}
    ]
})
~~~


@template:	api_config
@descr:

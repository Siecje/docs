dir
=============


@short: sets the direction of resizing 
	

@type: string
@example:
var resizeStick = new webix.ui.resizearea({
		container:"box",
		dir:"x",
        ...
});

@template:	api_config
@defined:	ui.resizearea	
@relatedapi:
	api/ui.resizearea_eventPos_config.md
    api/ui.resizearea_dir_config.md
    api/ui.resizearea_start_config.md
@descr:
The parameter can take only one of a predefined set of values. <br>
The predefined values are:

- **'x'** - horizontal resizing
- **'y'** - vertical resizing

getValue
=============

@short:
	returns the current value of the Barcode

@params:

@returns:

- value		string		the current Barcode value


@example:

var barcode = new webix.ui({
	view:"barcode",
    value:"123456789012",
});

barcode.getValue(); //returns "123456789012"

webWorker
=============


@short:
	sets the path to the web worker source file

@type: string
@example:
webix.ui({
	view:"pivot",
	webWorker:"../../codebase/pivot.worker.js",
});

@template:	api_config
@descr:

The file with the worker script is provided in the Pivot package.

cellWidth
=============

@short:
specifies the width of an item's cell for a chart in dynamic mode

@type: number
@default:30
@example:
webix.ui({
    id: "chart",
    view:"chart",
    type:"line",
    value:"#yValue#",
    dynamic: true,
    cellWidth: 50
});

@template:	api_config

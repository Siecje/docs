columnWidth
=============


@short: defines the default width for columns (in pixels)
@type: number

@example:
webix.ui({
	view:"datatable",
	columnWidth:150,
	...
});

@template:	api_config
@defined:	ui.datatable	
@related:
	datatable/columns_configuration.md
@descr:
The default value of the parameter is 100px.

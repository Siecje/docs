paddingY
=============

@short:
	sets the top and bottom padding (applies the specified value to both sides)

@type: number
@default:30 
@example:
webix.ui({
    view:"fieldset", 
    label:"Field Set 1",
    paddingY:25,
    body:{
        rows:[
            { view:"text", label:"Email"},
            { view:"text", label:"Password"}
        ]
    }
});


@template:	api_config

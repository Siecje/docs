getPath
=============

@short:
	gets an array of folders' ids in a path

@params:

- id			string			optional, the id of the selected folder 

@returns:
- pathIds		array			an array of folders' ids

@example:

var pathIds = fmanager.getPath();
 
// or
 
var pathIds = fmanager.getPath(id);

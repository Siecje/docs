inputAlign
=============

@short:
	the alignment of an input towards its container

@type: string
@example:
{view:"text", type:'password', value:'123pass', label:"Password", inputAlign:"right"}

@template:	api_config
@descr:
Possible values are "left" and "right". An input is left-aligned by default.

margin
=============


@short:defines the space around elements (applies the specified value to all elements)
	

@type:number 
@example:
webix.ui({
		container:"box",
		view:"toolbar", 
        margin:20,
		cols:[
			{ view:"button", value:"Load" },
			{ view:"button", value:"Save" },
            { view:"button", value:"Delete" }
		]
});

@template:	api_config

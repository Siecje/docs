dy
=============

@short: the top offset of the tooltip from the mouse pointer
	
@type: number
@example:
tooltip:{
	template:"#title# - #year#",
	dx: 30,
	dy: 5
}


@template:	api_config
@related:
	desktop/tooltip.md
@relatedapi:
	api/ui.tooltip_dx_config.md
@descr:
By default the parameter is equal to 0px. 

A negative value places the tooltip above the mouse pointer.

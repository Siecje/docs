optionIndex
=============


@short: index of the option with specified ID
	

@params:
- ID 		id		ID of the needed option

@returns:
- index		number		index of the option

@example:
$$("segmented1").optionIndex(1); 

@template:	api_method
@descr:

Since options are stored in the array, option indexes are numbered staring from 0.

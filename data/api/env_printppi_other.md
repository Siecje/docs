printPPI
=============

@todo: check links

@short:
	sets  default pixel per inch (ppi) ratio for printing

@type: number
@default: 96

@example:
webix.env.printPPI = 96;

@template:	api_config
@descr:

The option is used to define the default PPI used during [webix.print()](api/_print.md) call.

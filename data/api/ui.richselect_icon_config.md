icon
=============


@short: sets an icon type
	

@type: string
@example:
webix.ui({
	rows:[
		{ view:"icon", icon:"mail"},
		{ view:"icon", icon:"users"}
        ]
});

@template:	api_config
@related: 
	desktop/icon_types.md
@relatedsample: 
 	16_skins/03_icons.html
@descr:

Icons can be defined as 

- separate views;
- button types;
- CSS classes. 

Icon types are taken from [Font Awesome](http://fortawesome.github.io/Font-Awesome/).

footer
=============

@short:
	calculates the total sum per column

@type: string
@example:
webix.ui({
    view:"pivot",
    footer: "sumOnly",
    //...
});

@template:	api_config
@descr:
The "sumOnly" value of the *footer* property forces the summing function to process only columns that contain sums already.

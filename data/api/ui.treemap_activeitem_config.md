activeItem
=============


@short:
	specifies if childs of the branches will be rendered in the TreeMap

@default:false
@type: boolean
@example:
webix.ui({
    view:"treemap",
    activeItem: true,
    url: "data/data.json"
});

@template:	api_config
@descr:




@seolinktop: [popular javascript framework](https://webix.com)
@seolink: [TreeMap](https://webix.com/widget/treemap/)

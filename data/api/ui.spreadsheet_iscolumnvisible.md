isColumnVisible
=============



@short:
	checks whether the specified column is visible

@params:

- columnId		number		the column id

@returns:

- state			boolean		true, if the column is visible, otherwise - false

@example:

var isVisible = $$("ssheet").isColumnVisible(2);

onBeforeTabClick
=============


@short:
	fired before a user clicks a tab

@params:
- id		id		the id of the tabbar
- tab		id		the id of the clicked tab

@example: 
	
some.attachEvent("onBeforeTabClick", function(id, tab){
    //... some code here ... 
});

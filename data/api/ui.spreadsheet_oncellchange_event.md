onCellChange
=============


@short:
	fires when a cell is changed

@params:

- row			number			the number of the row 
- column		number			the number of the column 
- value			string			the value of the cell


@example:
$$("ssheet").attachEvent("onCellChange", function(row, column, value){
    // some code here
});

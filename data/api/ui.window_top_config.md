top
=============


@short:
	the top offset of a window

@type: number
@example:

webix.ui({
	view:"window",
    top:100,
    head:"..",
    body:{...}
}).show();

@template:	api_config
@defined:	ui.window	

@relatedapi:
	api/ui.window_left_config.md
    api/ui.window_position_config.md
@related:
	desktop/window.md
    desktop/popup.md
	
@descr:

If you set **position** and **left** or **top** at the same time - **left** and **top** parameters will be ignored.

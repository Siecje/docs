{{memo virtual scrolls}}

The component renders plain scrolls without content. 

### Constructor

~~~js
	var vscroll = webix.ui({
		view:"vscroll", 
		container:"mydiv", 
		...config options goes here..
	})
	//or, in case of jQuery
	$("#mydiv").webix_vscroll({
		...config options goes here..
	});
~~~

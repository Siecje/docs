### Constructor

~~~js
	var textarea = webix.ui({
		view:"textarea", 
		container:"mydiv", 
		...config options goes here..
	})
	//or, in case of jQuery
	$("#mydiv").webix_textarea({
		...config options goes here..
	});
~~~

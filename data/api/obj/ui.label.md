
{{memo a non-editable text control}}

The component is used to display static text on a page. Check [label](desktop/label.md) documentation for more detailed description.

### Constructor

~~~js
var label = webix.ui({ 
	view:"label", 
	label: "Label", 
	inputWidth:100, 
	align:"left"
});
~~~

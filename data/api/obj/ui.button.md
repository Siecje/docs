
{{memo a clickable button}}

The control has a collection of preset looks and can be used both as a stand-alone view and as a part of a form or toolbar. Fully customizable. Check [button](desktop__controls.md#button) documentation for more detailed description.

### Constructor

~~~js
	var button = webix.ui({
		view:"button", 
		container:"mydiv", 
		...config options goes here..
	})
	//or, in case of jQuery
	$("#mydiv").webix_button({
		...config options goes here..
	});
~~~

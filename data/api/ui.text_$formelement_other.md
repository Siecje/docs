$formElement
=============


@short: flag which instruct to process such view as form input
	

@type: bool

@example:

@template:	api_config
@descr:

Form will try to clear view which have set $formElement flag on data reloading.  
Also during data binding, form will fill such inputs with data. 

For custom code - there is no any sane scenario, where you will need to set this field manually.

tooltip
=============


@short: sets a popup message appearing on pointing a mouse cursor over the dedicated item.
	

@type: object
@example:
webix.ui({
            view:"dataview",
            ...
            },
            tooltip:{
                Rating: #rating#<br/>
                Votes: #votes#"
            }
        });

@template:	api_config

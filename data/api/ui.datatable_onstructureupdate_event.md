onStructureUpdate
=============


@short: fires when structure of datatable was changed ( column added, hidden or reconfigured ) 
	

@params:

@example:

mydata.attachEvent("onStructureUpdate", function(){
    webix.message("Grid was updated");
})

@template:	api_event

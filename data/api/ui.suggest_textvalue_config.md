textValue
=============


@short: defines which field from suggest list will be used as text after selecting an option
	

@type: string
@example:
webix.ui({
    view:"select",
    options:{
       templateValue:"name",
       data:[
          { name:"Mark", city:"Hrodno" },
          { name:"Anna", city:"Brest" }
       ]
    }
})

@template:	api_config
@descr:

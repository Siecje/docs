timeFormat
=============


@short: sets the default locale time format
@type: string

@example:

webix.i18n.timeFormat = "%H:%i";
webix.i18n.setLocale();

@template:	api_config
@defined:	i18n	
@descr:


You need to call <a href="api/i18n_setlocale.md">setLocale</a> to apply the new format settings

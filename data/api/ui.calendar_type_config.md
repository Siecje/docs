type
=============

@short: type of calendar view

@values:
- time		switches calendar to time view	
- month		switches calendar to month view
- year		switches calendar to year view
	

@type: string
@example:

webix.ui({
	view:"calendar",
    type:"time"
});

@template:	api_config

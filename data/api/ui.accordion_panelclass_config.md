panelClass
=============


@short: returns the name of a css class applied to the accordion panels
	

@type: string
@example:
var class = $$('myaccordion').config.panelClass;

@template:	api_config

@related:
	desktop/accordionitem.md
    desktop/accordion_css.md 
@descr:
The default value is "accordionitem".

scroll
=============


@short:the direction of scrolling
	

@type:string
@example:
webix.ui.vscroll({
		container:"scroll_here",
		scrollHeight:500,
		scroll:"y"
});

@template:	api_config
@defined:	ui.vscroll	
@descr:
The parameter can take only one of a predefined set of values.<br>
The predefined values are:

- **'y'** - vertical scrolling
- **'x'** - horizontal scrolling (the default value)

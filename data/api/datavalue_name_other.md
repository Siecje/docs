name
=============


@short:indicates the name of the component (a read-only property)
	

@type:string

@example:
var data = new webix.DataValue({});
data.setValue("someValue");

var name = data.name; // returns 'DataValue'
@template:	api_config
@defined:	DataValue	
@descr:

The hardcoded value of the property is "DataValue".

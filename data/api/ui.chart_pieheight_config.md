pieHeight
=============


@short: the height of the 3d pie
	

@type: number
@example:
webix.ui({
	view: "chart",
	type:"pie3D",
	pieHeight:50,
    ...
});

@template:	api_config
@related: 
	desktop/pie_charts.md
@descr:
{{note
The property is applicable only to **pie3D** charts
}}

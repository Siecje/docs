getList
=============



@short: returns inner data control from suggest
	

@returns:
- view     WebixListView     inner data control
	

@example:

var list = $$("suggest").getList();


@template:	api_method
@descr:


For different suggests it will return different views ( list for default one,
calendar for datepicker and etc. ) 



@seolinktop: [javascript ui framework](https://webix.com)
@seolink: [list in javascript](https://webix.com/widget/list/)

padding
=============


@short: defines the space between the element borders and content (applies the specified value to all sides).
	

@type:number
@example:
webix.ui({
		container:"box",
		view:"toolbar",
        padding:10,
        cols:[
			{ view:"button", value:"Load" },
			{ view:"button", value:"Save" },
			{ view:"button", value:"Delete" }
		]
});

onResize
=============


@short:occures when the user moves the 'resize' cursor
	

@params:
- pos  number  the top (<i>vertical resizing</i>) or left (<i>horizontal resizing</i>) relative position of the cursor (relative to the parent container)

@example: 
	
some.attachEvent("onResize", function(pos){
    //... some code here ... 
});

@template:	api_event

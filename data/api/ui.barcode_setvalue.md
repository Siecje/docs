setValue
=============

@short:
	sets a value for Barcode

@params:

- value			string			a new value for Barcode 


@example:

var barcode = new webix.ui({
   view:"barcode",
   value: "123456789012"
});
// set the new value
barcode.setValue("453456789015");

@template:	api_method
@descr:

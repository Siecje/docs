onBeforeCursorChange
=============


@short: fires before the cursor changes its position.
	

@params:
-	id	string	the id of an item that the cursor is set on (the old cursor position).

@example: 
	
some.attachEvent("onBeforeCursorChange", function(id){
    //... some code here ... 
});

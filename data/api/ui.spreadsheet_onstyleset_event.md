onStyleSet
=============


@short:
	fires when a new style is set for a cell

@params:
- name			string		the set style name
- value			string		the cell value

@example:
$$("ssheet").attachEvent("onStyleSet",function(name, value){
	// your code here
});

@template:	api_event

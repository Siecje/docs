filterWidth
=============

@short:
	sets the width of filters (both input and label) in the pivot header

@type: number
@example:
webix.ui({
    view:"pivot-chart",
    filterWidth: 250
});

@template:	api_config
@descr:
The default value is 300.

desktop/pivotchart.md#initialization

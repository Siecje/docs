onFilterCreate
=============
    
@short:
	fires each time a filter is created 

@params:
- props		object 		filter's properties
- config	object		config of the input view used for filter 


@example:
$$("pivot").attachEvent("onFilterCreate", function(props, config){
	// your code
});

@template:	api_event

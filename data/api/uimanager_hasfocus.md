hasFocus
=============

@short:
	checks whether the specified view has focus

@params:
- id		id          the ID of the view to check

@returns:
- result		boolean       <i>true</i>, the view is in focus. Otherwise, <i>false</i>.
	

@example: webix.UIManager.hasFocus($$('books'));

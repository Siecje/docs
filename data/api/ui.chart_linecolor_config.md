lineColor
=============


@short:
	sets the color for strips dividing the pie sectors 

@type: string
@example:
webix.ui({
	view: "chart",
	type:"pie",
	lineColor:"#A5A5A5",
    ...
})
                            

@template:	api_config
@defined:	ui.chart	
@related: 
	desktop/pie_charts.md
@descr:
{{note
The property is applicable only to **pie** charts
}}

The default value of the property is "#ffffff".

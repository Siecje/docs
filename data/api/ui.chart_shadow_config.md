shadow
=============


@short:
	enables/disables the pie shadow

@type: bool
@example:
webix.ui({
	view: "chart",
	type:"pie",
    shadow:0,
    ...
})

@relatedsample:
	08_chart/04_pie_chart/02_3d_chart.html
@template:	api_config
@defined:	ui.chart	
@descr:
{{note
The property is applicable only to **pie** charts
}}

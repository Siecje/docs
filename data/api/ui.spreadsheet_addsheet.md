addSheet
=============


@short:
	adds a new sheet into a SpreadSheet

@params:

- content		object		an object with the sheet's content



@example:
// get content of an active sheet
var content = $$("ssheet").serialize();
// copy to a new sheet
$$("ssheet").addSheet(content);

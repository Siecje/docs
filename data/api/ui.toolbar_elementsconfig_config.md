elementsConfig
=============


@short: settings, which will be applied to all nested inputs
	

@type: hash
@example:

webix.ui({
    view:"form",
    elements:[
        {view:"text", label:"First Name"},
        {view:"text", label:"Last Name"}
    ],
    elementsConfig:{
        labelPosition:"top"
    }
});

@template:	api_config
@descr:

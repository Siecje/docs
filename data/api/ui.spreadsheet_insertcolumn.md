insertColumn
=============


@todo:
	check 

@short:
	inserts an empty column to the left of the specified one

@params:

- columnId			number		the column id



@example:

$$("ssheet").insertColumn(columnId);

@template:	api_method
@descr:
If the column id isn't specified, a new column will be inserted before the selected one.

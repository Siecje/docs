addCss
=============


@short: applied CSS class to a component item


@params:	
- id	string,number	ID of the neccessary item
- CSS class		string 		CSS class name

@example:
var menu = $$("menu1").getSubMenu(id);
menu.addCss(id, "webix_disabled");

subMenuPos
=============


@short: aligns submenu towards main menu

@type: string
@example:
webix.ui({
	view:"menu",
	width:150,
	subMenuPos:"right"
});

@template:	api_config

@related:
	desktop/menu.md
@relatedsample:
	03_menu/01_menubar.html
@descr:
The following values are supported:

- **right**;
- **left**.

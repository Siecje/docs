onResizeEnd
=============


@short:occures when the user ends moving the 'resize' cursor (releases the mouse button)
	

@params:
- pos  number  the top (<i>vertical resizing</i>) or left (<i>horizontal resizing</i>) relative position of the cursor (relative to the parent container)

@example: 
	
some.attachEvent("onResizeEnd", function(pos){
    //... some code here ... 
});

@template:	api_event

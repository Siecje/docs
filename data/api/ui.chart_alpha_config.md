alpha
=============


@short:
	defines the opacity of chart items

@type: number
@example:
webix.ui({
	view:"chart",
    container:"chartDiv",
	type:"bar",
    ...
	alpha:0.5
});

@template:	api_config
@relatedsample:
	08_chart/02_color/01_custom.html
@related: 
	desktop/bar_charts.md
@defined:	ui.chart	
@descr:
The parameter can take value from 0.0 (fully transparent) to 1.0 (fully opaque).<br>
The default value - 1.

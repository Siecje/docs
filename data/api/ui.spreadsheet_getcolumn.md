getColumn
=============


@short:
	returns the configuration object of the specified SpreadSheet column

@params:

- id		string		the column id

@returns:
- config		object		the column's configuration object


@example:
var config = $$("ssheet").getColumn(id);

@template:	api_method
@descr:

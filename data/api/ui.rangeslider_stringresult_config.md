stringResult
=============


@short:
	if the parameter is set to true, the method getValue returns data as string (otherwise, as object)
@type:boolean

@example:
{ view:"rangeslider", stringResult:true, format:"%d  %M %Y" },

@template:	api_config
@defined:	ui.rangeslider	

@descr:

The default value is <i>false</i>.

datatype
=============


@short:
	the type of serverside response 

@type: string
@example:
{
	view:"uploader",
	id: "uploader_1",
	value:"Upload file",
	link:"mylist",
	upload:"php/upload.php",
	datatype:"json"
}

@template:	api_config
@descr:
The property is optional. JSON format is set by default.

dateFormat
=============

@short: sets the default locale date format
@type: string

@example:

webix.i18n.dateFormat = "%d-%m-%Y";
webix.i18n.setLocale("en-US");

@template:	api_config
@defined:	i18n	
@descr:

You need to call <a href="api/i18n_setlocale.md">setLocale</a> to apply the new format settings

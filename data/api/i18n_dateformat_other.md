dateFormat
=============

@short: sets the default locale date format
@type: string

@example:

webix.i18n.dateFormat = "%d-%m-%Y";
webix.i18n.setLocale();

@template:	api_config
@defined:	i18n	
@descr:

You need to call api/i18n_setlocale.md to apply the new format settings.

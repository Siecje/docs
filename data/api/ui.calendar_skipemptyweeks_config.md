skipEmptyWeeks
=============


@short:
	enables/disables displaying empty weeks (true)

@type: bool
@example:


@template:	api_config
@defined:	ui.calendar	
@related: 
	desktop/calendar.md
@descr:

Defines whether 6 weeks (42 days) must be displayed all the time or weeks which completely belong to the next month should be skipped.

labelOffset
=============


@short:
	the offset of sector labels from the pie frame 

@type: number
@example:
webix.ui({
	view: "chart",
	type:"pie",
    label:"#month#",
	labelOffset:40,
    ...
})

@relatedapi:
	api/ui.chart_label_config.md
@related: 
	desktop/pie_charts.md
@template:	api_config
@defined:	ui.chart	
@descr:
The default value of the property is 20.

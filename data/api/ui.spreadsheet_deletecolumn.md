deleteColumn
=============


@short:
	deletes a column by id

@params:
- columnId		number		the column id


@example:

$$("ssheet").deleteColumn(columnId);

@template:	api_method
@descr:
If the columnId parameter isn't specified, the method deletes the selected column.

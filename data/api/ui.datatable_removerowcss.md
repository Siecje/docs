removeRowCss
=============


@short:
	remove css class from the row	

@params:
- id	string		row id
- css_name	string		name of css class to remove


@example:

$$("mytable").removeRowCss(152, "some_style");

@template:	api_method
@descr:

Works as short alias for removeMark command

movePortlet
=============

@short:
	moves portlet to the marked area

@params:

- target		string		id of the view where the dragged portlet will be dropped
- mode			string		the mode of dropping ("top","bottom","left","right")



@example:

$$("myportlet").movePortlet(target,mode);

@template:	api_method
@descr:

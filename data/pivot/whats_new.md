Pivot What's New
=================

###Version 4.2

New Features
-------------

- ability to [offload work to a web worker](pivot/webworkers.md)
- api/ui.pivot_filterplaceholder_config.md property to set placeholder for filters added
- api/ui.pivot_onfiltercreate_event.md event
- "fields tree", datepicker, filter demos added

Updates
-------

- structure of the external data source is simplified
- optional grouping fields as Tree

Fixes
--------

- onHeaderInit event does not fire
- filters bug with values
- order of operations in header
- default filter in visual dialog when "select" filter was disabled
